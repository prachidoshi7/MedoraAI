# Planning

Use this folder for project-level planning and design documents.

## Current Documents

- `project-abstract.md` - hackathon/project abstract and team/project summary.
- `product-requirements.md` - product requirements, goals, users, and demo success criteria.
- `technical-design.md` - architecture, API design, ML pipeline, and implementation plan.

## Suggested Additions

- `timeline.md` - milestone plan and deadlines.
- `scope.md` - MVP scope, stretch goals, and out-of-scope items.
- `risk-register.md` - risks, owners, mitigation, and current status.
