# Setup Guide

Update this file when the application code is added.

## Prerequisites

- Git
- Node.js, if the frontend uses JavaScript or TypeScript
- Python, if the backend or ML pipeline uses Python
- Docker, if the team chooses containerized setup

## Local Setup

1. Clone the repository.
2. Install frontend dependencies.
3. Install backend dependencies.
4. Configure environment variables.
5. Start frontend, backend, and ML services.

## Notes

- Add exact commands once `frontend`, `backend`, and `ml` contain runnable source code.
- Document required API keys in `.env.example`, not in this file.
